프로젝트 개요

01
게임 컨셉

필리핀 고유 신화(Bathala, Apolaki, Mayari, Bakunawa 등)를 세계관으로 삼은 모바일 수집형 RPG. 완전 실시간 조작 MMORPG가 아닌 덱 구성 → 자동/반자동 전투 → 보상 수집 → 강화 → 가챠 루프 구조로 설계하여 1인 개발이 현실적으로 가능한 범위로 기획.

💡 화면 방향은 가로 고정 (16:9) — 전투 몰입감, 스킬 배치 공간 확보. 전투 조작은 설정에서 Full Auto / Semi-Auto / Full Manual 선택 가능.
장르 선택 근거
비교 항목실시간 조작 MMORPG수집형 RPG (본 기획)
개발 난이도매우 높음 (서버 동기화, 핵 방지)낮음 (결과값만 서버 처리)
서버 비용높음Supabase로 충분
수익화 적합성보통 (실력이 과금 대체 가능)높음 (컬렉션 욕구 무한)
필리핀 시장 적합성높음매우 높음 ✓
모바일 Top10 점유일부대부분 ✓

레이아웃 구성
  • 퀘스트 사이드바 — 진행 중인 퀘스트 3종 프로그레스 바 표시
  • SVG 연결선 — 지역 간 점선 연결, 진입 가능/불가 시각 구분
  • 지역 오브 — 속성별 색상 오라 + 플로팅 애니메이션
  • 잠긴 지역 노출 — "저기 가려면 강해져야 해" 동기 부여
  • 현재 위치 표시 — Gitnang Daigdig 빨간 테두리 강조
🗓 퀘스트 진행도를 사이드바에 항상 표시하여 "오늘 해야 할 것"이 명확. 일일 미션 완료 시 보상 팝업 → 자연스러운 에너지 소진 → 충전 과금 유도.
